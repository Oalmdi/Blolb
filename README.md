# Blob 

## Overview
Blob Memory Manager is a simple C project that simulates memory management using a block, range, and cursor abstraction. It allows creating a memory block, navigating through it, and reading/writing data safely within defined boundaries.

## Features
- Create and delete memory blocks.
- Read and write bytes at specific offsets.
- Define memory ranges.
- Move and jump using a cursor.
- Combine Block, Range, and Cursor into a Blob object.
- Print the current memory contents and blob status.
